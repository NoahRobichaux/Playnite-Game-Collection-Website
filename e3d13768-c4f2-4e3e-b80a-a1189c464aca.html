Description

"Let's begin your own story."
This is a text-based idle RPG game.
There are already more than 3,000 unique story actions and it's still growing.

Game Features
- Action buttons to select what to do and enhance abilities
- Automated battles in Dungeon
- A system to summon monsters and strengthen them
- Skill and Equipment for summoned monsters
- An upgrade system called Routine with offline bonus
- A system to automate Action and Dungeon
Other features can be unlocked as you progress through the game.

System Features.
- A multi-ending system where the ending changes depending on your choices.
- Offline bonus allows the progress made when the game is closed to be reproduced when the game is launched.

Story
You'll start this game from the village.
Your father says "Be strong, and save your mother.".
Clear many quests and battle against monsters to be strong.
Making friends may be good for your journey.
Everything starts from here.

Controls
Click the buttons to select action.
Read the help section in game for detail.
